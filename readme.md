## DT208G - programmering i TypeScript

Det här är ett projekt för kursen programmering i typescript, VT 2026.

Webbplatsen är publicerad på Netlify: https://dt208g-1-sini2500.netlify.app/

---

Lab 1 går ut på att använda TypeScript för att bygga en lokal app för kurshantering.

Vite används för bygget, och koden är enkel nog att få plats i en main.ts fil.

Formulär för input.

Select-fält för att välja progression, enklaste lösningen, med lite extra validering i TS.

Varje kurs presenteras i ett litet kort.

Funktioner för att ladda, spara, lägga till och ta bort kurser från localstorage. 

Funktioner för att kontrollera om input är korrekt och unik.
